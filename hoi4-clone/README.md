# Hearts of Iron 4 - Simple Clone

Упрощённая браузерная версия Hearts of Iron 4, созданная на чистом HTML/CSS/JavaScript.

## Возможности

### Игровые механики
- **Карта мира** с основными странами (Германия, Франция, Великобритания, СССР, США, Италия, Япония)
- **Система ресурсов**: политическая власть, заводы, верфи, дивизии
- **Производство**: строительство гражданских/военных заводов и верфей
- **Военное производство**: обучение дивизий и производство вооружения
- **Дипломатия**: объявление войны и улучшение отношений
- **Временная шкала**: начало в 1936 году с управлением скоростью игры
- **Система событий**: лог всех важных игровых действий

### Управление
- **Пауза/Возобновление** игры
- **Скорость игры**: обычная, быстрая, очень быстрая
- **Строительство**: требует политической власти
- **Военные действия**: объявление войны другим странам

## Как играть

1. Откройте `index.html` в браузере
2. Вы начинаете играть за Германию в 1936 году
3. Используйте политическую власть для:
   - Строительства заводов (50 ПВ)
   - Обучения дивизий (20 ПВ)
   - Дипломатических действий (30-100 ПВ)
4. Следите за ресурсами и развивайте свою державу
5. Объявляйте войны и расширяйте влияние

## Технологии
- HTML5 Canvas для отрисовки карты
- Vanilla JavaScript (без внешних библиотек)
- CSS3 для современного UI

## Особенности реализации
- Объектно-ориентированная архитектура
- Игровой цикл с поддержкой разных скоростей
- Визуализация состояния войны на карте
- Цветовая индикация стран и их статуса
- Система логирования событий
